# Input

A styled text input field that supports all native HTML input types.

## Import

```tsx
import { Input } from '@edge/ui'
```

## Usage

```tsx
<Input placeholder="Enter a value..." />
```

## Types

```tsx
<Input type="text" placeholder="Text" />
<Input type="email" placeholder="Email" />
<Input type="password" placeholder="Password" />
<Input type="number" placeholder="Number" />
<Input type="search" placeholder="Search" />
```

## With label

```tsx
import { Label } from '@edge/ui'

<div className="flex flex-col gap-1.5">
  <Label htmlFor="username">Username</Label>
  <Input id="username" placeholder="edge-user" />
</div>
```

## Disabled

```tsx
<Input disabled placeholder="Disabled input" />
```

## File input

```tsx
<Input type="file" />
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `string` | `'text'` | HTML input type |
| `placeholder` | `string` | — | Placeholder text |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

All standard HTML `input` attributes are supported.
