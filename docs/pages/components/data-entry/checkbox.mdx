# Checkbox

A controlled checkbox input with custom styling.

## Import

```tsx
import { Checkbox } from '@edge/ui'
```

## Controlled

```tsx
import { useState } from 'react'

function ControlledCheckbox() {
  const [checked, setChecked] = useState(false)
  return (
    <Checkbox
      checked={checked}
      onCheckedChange={setChecked}
    />
  )
}
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes (applied to the visual indicator) |

All standard HTML `input` attributes (except `checked` and `onCheckedChange`) are also supported.
