# Button

A button component with multiple variants and sizes, built on HTML `<button>` with support for links via `asChild`.

## Import

```tsx
import { Button, buttonVariants } from '@edge/ui'
```

## Usage

```tsx
<Button>Default</Button>
<Button variant="destructive">Delete</Button>
<Button variant="outline">Cancel</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

## Sizes

```tsx
<Button size="sm">Small</Button>
<Button size="default">Default</Button>
<Button size="lg">Large</Button>
<Button size="icon"><Icon /></Button>
```

## With Icon

```tsx
import { Plus } from 'lucide-react'

<Button>
  <Plus className="mr-2 h-4 w-4" />
  Create Resource
</Button>
```

## As Link

Use the `asChild` prop to render as a different element:

```tsx
import Link from 'next/link'

<Button asChild>
  <Link href="/dashboard">Go to Dashboard</Link>
</Button>
```

## Disabled

```tsx
<Button disabled>Disabled</Button>
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | `'default'` | Visual style variant |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon'` | `'default'` | Button size |
| `asChild` | `boolean` | `false` | Render as child element (uses Radix Slot) |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

All standard HTML button attributes are also supported.

### buttonVariants

Utility function for applying button styles to non-button elements:

```tsx
import { buttonVariants } from '@edge/ui'
import Link from 'next/link'

<Link href="/new" className={buttonVariants({ variant: 'outline' })}>
  Create New
</Link>
```
