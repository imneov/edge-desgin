# Icon Gallery

Edge Design System uses [Lucide React](https://lucide.dev) as the icon library. Below are the icons used across Edge Console, grouped by category.

|------|--------|-------|
| `LayoutDashboard` | `import { LayoutDashboard } from 'lucide-react'` | 仪表盘入口 |

| `ChevronRight` | `import { ChevronRight } from 'lucide-react'` | 面包屑、折叠展开 |

## Standard Usage

```tsx
import { Plus, RotateCw, MoreHorizontal } from 'lucide-react'

// 固定尺寸（表格行操作）
<MoreHorizontal className="h-4 w-4" />

// 带间距（按钮图标）
<Plus className="mr-2 h-4 w-4" />

// 旋转动画（加载中）
<Loader2 className="h-4 w-4 animate-spin" />
```
