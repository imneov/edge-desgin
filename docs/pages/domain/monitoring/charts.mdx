# Charts

Time-series area chart for resource usage visualization.

## ResourceChart

A generic time-series area chart from `@edge/ui`. Displays percentage usage, a progress bar, and an area chart.

### Import

```tsx
import { ResourceChart } from '@edge/ui'
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Chart title |
| `currentValue` | `string` | — | Current value string (e.g. `"2.4 / 4 cores"`) |
| `percentage` | `number` | — | Usage percentage for progress bar and display |
| `timeSeries` | `ResourceChartDataPoint[]` | — | Time-series data array |
| `color` | `string` | `"#3b82f6"` | Chart theme color (hex) |
| `unit` | `string` | `"%"` | Unit label for axis and tooltip |
| `showGrid` | `boolean` | `true` | Show chart grid lines |
| `className` | `string` | — | Additional CSS classes |

**ResourceChartDataPoint Type:**

```tsx
interface ResourceChartDataPoint {
  time: string | number  // time label for X axis
  value: number          // numeric value for Y axis
}
```

### Progress Bar Color Rules

The progress bar color changes automatically based on `percentage`:

| Range | Color |
|-------|-------|
| `> 80%` | Red (`bg-red-500`) |
| `> 60%` | Yellow (`bg-yellow-500`) |
| `≤ 60%` | Blue (`bg-blue-500`) |

### Usage in Edge Console

In `edge-console`, `ResourceChart` is used in the telemetry dashboards for CPU, memory, and disk metrics. Data comes from the monitoring service SDK hooks:

```tsx
import { ResourceChart } from '@edge/ui'

function CpuChart({ timeSeries, currentUsage, percentage }) {
  return (
    <ResourceChart
      title="CPU 使用率"
      currentValue={currentUsage}
      percentage={percentage}
      timeSeries={timeSeries}
      color="#3b82f6"
      unit="%"
    />
  )
}
```
